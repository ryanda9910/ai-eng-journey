# ai-eng-journey

Daily build-in-public log of a 16-week AI engineer challenge. Blog via GitHub Pages
(`_posts/`), social media images hosted in `shots/`.

Phases: LLM engineering (W1-3) → production RAG (W4-8) → agents + failure handling
(W9-13) → portfolio (W14-16). Every deliverable ships live with eval numbers and is
graded by an independent verifier agent.

Automation lives in `~/project/ai-eng-challenge/` (not in this repo).
